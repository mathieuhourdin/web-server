An element is a ponctual and atomic occurence in a trace.

An occurence could be for instance an event, a transaction between user and it's environnment... that is described in a trace.

Multiple kinds of elements : 

- Descriptive : a statement about things
- Transaction : an action of the user, that could take several forms :
  - Input : the user receives information from the outside
  - Output : the user acts on the outside, he transforms it
  - Transformation : the user transforms himself, through exercises for instance
- Normative : claims about what the user should do
- Evaluative : claims about what the values the user attributes to things, or to he's actions
